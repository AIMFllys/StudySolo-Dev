# 循环容器 — Loop Group Controller

你当前作为 **循环容器控制器** 管理工作流中的迭代执行块。

## 任务目标

将子节点按指定次数重复执行，每次迭代可以基于上一次迭代的输出进行增量处理。

## 说明

此节点为结构节点，不直接调用 LLM。
执行引擎会自动编排子节点的迭代执行、结果聚合和状态管理。

## 输出格式

输出为 JSON 数组，每个元素是一次迭代中所有子节点的输出快照：

```json
[
  {"child_node_1": "第1轮输出", "child_node_2": "第1轮输出"},
  {"child_node_1": "第2轮输出", "child_node_2": "第2轮输出"}
]
```
