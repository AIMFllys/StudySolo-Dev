# 快速开始

欢迎来到 StudySolo！这篇指南会带你完成一次完整体验：从输入学习目标，到生成工作流，再到运行并查看结果。你不需要先理解所有节点，只要先跑通一个最小流程，就能快速判断 StudySolo 是否适合你的学习任务。

## 前置条件

在开始之前，请确保你已：

- 注册并登录 StudySolo 账号
- 完成邮箱验证
- 登录后进入工作台
- 准备一个明确的学习目标

如果你只是想体验，也可以先使用下面这个示例目标：

```text
我想在 4 周内入门 Python 数据分析。
我有 Python 基础，但不熟悉 Pandas、可视化和项目实践。
请帮我生成一个每周学习计划，并包含练习、复盘和最终项目。
```

## 创建第一个工作流

### 步骤 1：新建工作流

点击工作台左侧的「新建工作流」按钮，输入你的学习目标：

```
我想学习 Python 数据分析，为期 3 个月
```

不要只写关键词，最好同时说明时间、基础、目标和偏好的输出形式。

| 信息 | 示例 | 作用 |
| --- | --- | --- |
| 学习主题 | Python 数据分析 | 告诉 AI 要处理的领域 |
| 当前基础 | 会基础语法，不熟 Pandas | 避免生成过难或过浅的计划 |
| 时间限制 | 4 周，每天 1 小时 | 让计划更符合现实节奏 |
| 结果要求 | 每周任务、练习、复盘问题 | 让输出更可执行 |

### 步骤 2：AI 生成学习路径

点击「生成工作流」按钮，AI 将：

1. 分析你的学习目标
2. 拆解知识点结构
3. 生成完整的学习节点图

如果生成结果不够具体，不需要重新开始。你可以继续用 AI 对话补充要求：

```text
请把每周计划拆到每天，并为每一天增加一个可检查的完成标准。
```

### 步骤 3：查看节点图

生成完成后，你将看到：

- **输入节点**：接收学习目标
- **AI 分析节点**：拆解知识框架
- **规划节点**：生成学习路径
- **资源节点**：推荐学习资料
- **输出节点**：完整学习计划

先不要急着运行，建议按下面顺序检查：

1. 是否有明确的输入节点
2. 是否先分析目标，再生成计划
3. 是否有资源、练习或评估节点
4. 最终输出节点是否能回答你的原始需求

### 步骤 4：运行工作流

点击「运行」按钮，AI 将执行整个学习规划流程，最终输出：

- 阶段性学习目标
- 推荐学习资源
- 预计完成时间

运行时重点观察三件事：

- 哪个节点正在执行
- 每个节点输出了什么中间结果
- 最终结果是否足够具体、可执行、可复盘

如果最终结果太泛，可以从运行记录里找到最早出现问题的节点。通常不是最后一个输出节点的问题，而是前面的目标分析或约束描述不够清楚。

## 优化第一次结果

第一次生成的工作流可以看作草稿。你可以按目标继续优化：

| 问题 | 调整方式 |
| --- | --- |
| 计划太宽泛 | 要求每个阶段包含任务、产出和验收标准 |
| 难度不合适 | 补充自己的基础、薄弱点和可投入时间 |
| 资源太多 | 限定资源数量和语言，例如“每周最多 3 个资源” |
| 缺少练习 | 增加练习节点、测验节点或复盘节点 |
| 输出格式混乱 | 指定表格、清单或 Markdown 大纲 |

## 推荐的第一次工作流结构

如果你想手动搭一个最小可用流程，可以从下面这条线开始：

```text
学习目标输入 → 目标分析 → 知识拆解 → 周计划生成 → 练习设计 → 最终学习计划
```

这个结构足够简单，也覆盖了大多数个人学习场景。等你熟悉后，再加入条件分支、资源筛选、阶段测验和复盘总结。

## 常见问题

### Q: 生成的节点图可以修改吗？

可以。你可以手动拖拽调整节点位置，添加或删除节点，自定义学习路径。

### Q: 支持哪些类型的学习目标？

支持各类技能学习，包括编程语言、专业课程、考证准备、兴趣爱好等。

### Q: 一次要不要生成很复杂的工作流？

不建议。第一次先生成 5 到 8 个节点即可。复杂工作流更适合在跑通基础流程后逐步加节点，否则调试成本会明显上升。

### Q: 如果 AI 理解错了目标怎么办？

回到输入或分析节点，补充背景和限制条件。比起反复重新生成，直接修正关键节点通常更快。
