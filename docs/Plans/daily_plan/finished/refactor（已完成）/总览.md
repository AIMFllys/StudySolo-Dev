## 各阶段核心交付物

| 阶段 | 交付物 | 用户可感知变化 |
|------|--------|---------------|
| P0 | 类型安全、废代码清除 | 无直接可见变化 (稳定性提升) |
| P1 | 全新 UI 审美 | 视觉全面升级 |
| P2 | 节点面板 + 拖拽 + 配置 | **用户可手动搭建任意工作流** |
| P3 | 撤销/重做 + 崩溃恢复 | 操作可撤销 + 崩溃恢复 |
| P4 | 双模式面板 + AI 思考流 UI | 右侧面板可切换查看 AI 过程 |
| P5 | 流式节点生成 | **一句话创建工作流 + 全程可见** |
| P6 | 测试 + 监控 + 文档 | 长期质量保障 |

## 关键原则

1. **追加式开发**: 每个阶段的改动不破坏现有功能
2. **降级兼容**: P5 的流式生成失败可自动回退同步模式
3. **手动优先**: 先确保手动搭建可用 (P2)，再增强 AI 生成 (P5)
4. **每阶段可验证**: 每个阶段都有独立的验收标准
