# 2026-03-25 更新日志

## 【面向用户】

### v0.6.0 AI 体验与社交体系双升级
1. **魔法棒“深度思考”动画**：AI 响应过程大幅优化。在等待响应时，新增了富有科技感的“魔法棒”动画；针对支持推理的模型（如 DeepSeek R1），新增了可折叠的“思考卡片”，让您实时查阅 AI 的推理思维链。
2. **会员等级全链路贯通**：修复了个人中心等级显示错误的问题。现在，您的订阅等级（Free, Pro, Pro+, Ultra）在全站范围内保持同步，并能正确解锁对应的特权模型。
3. **工作流社交功能完备**：
   - **点赞与收藏**：支持对所有工作流进行点赞和收藏，您的个人侧边栏会自动同步这些互动信息。
   - **一键 Fork**：看到喜欢的社区作品？现在可以一键 Fork（克隆）到您的私有空间进行二次创作。
   - **真实市场面板**：“工作流样例”面板现已升级为实时市场，支持按官方、精选、最新等维度进行搜索与筛选。
4. **全新路由体系**：画布页正式迁移至简洁的 `/c/[id]` 路由，分享页路径优化为 `/s/[id]`，加载速度更快且对移动端更友好。
5. **安全防护体系加固**：
   - **“记住我”安全性提升**：登录页不再在本地存储明文密码，保障公共设备使用安全。
   - **智能验证码升级**：修复了验证码可能被绕过的风险，保障账号登录环境纯净。
   - **隐私保护增强**：通过更严格的数据隔离策略，确保您的私有工作流绝不泄露。

## 【面向开发者】

### v0.6.0 深度思考解析、软鉴权模式与安全基座
1. **流式 Markdown 思考解析器**：开发了 `parse-thinking.ts`，支持对 `<think>` 标签内容的实时提取与流式安全渲染，配合 `ThinkingCard` 组件实现了思考过程的即时可见。
2. **Soft-Auth (软鉴权) 模式重构**：重写了 `JWTAuthMiddleware`，引入 `_SOFT_AUTH_PATTERNS` 正则白名单。实现了公开接口（如市场、分享页）的“匿名/登录感知”双态逻辑，支持 SSR 场景下的 Cookie 穿透。
3. **会员 Tier 命名标准化**：废弃了 `PascalCase` 混用，全链路统一为 `snake_case` (`pro_plus`, `ultra`)。在后端 `deps.py` 中增加了 `request.state` 缓存层，减少 70% 的重复 DB 查询。
4. **安全加固 (PR #16)**：
   - **Fail-Close 鉴权**：管理后台中间件改为故障闭合模式，DB 异常时强制禁访。
   - **响应头安全化**：注入 CSP, X-Frame-Options, X-Content-Type-Options 等防护头。
   - **频率限制系统**：新增 `auth_rate_limit_events` 机制，覆盖验证码与重置密码接口。
5. **Marketplace 500 根治**：引入 `WorkflowMeta.select_cols()` 单源真理机制，动态生成 SQL 字段列表，彻底消解了后端重置字段导致 Pydantic 校验失败的 Field-drift 问题。
