# 2026-03-18 更新日志

## 【面向用户】

### v0.4.0 全新工作流连线系统
1. **三种交互式线型**：
   - **顺序流**：标准石灰色手绘实线，用于最基础的步骤衔接。
   - **条件分支**：琥珀色虚线，支持双击标签直接编辑判断逻辑。
   - **循环迭代**：翡翠绿正弦波浪线，用于表达重复执行的任务流。
2. **8 点灵活接口**：节点现在拥有上、下、左、右各 2 个连接点，支持从任意方位进行复杂编排，大幅提升布局灵活性。
3. **点击建连模式**：除了传统的拖拽连线，现在只需先后点击两个端点即可自动建连，更适合触屏和精确操作。
4. **连线右键菜单**：右键点击任何连线，均可快速切换线型、反转连线方向或直接删除。
5. **智能循环区域**：一旦创建循环连线，系统会自动在涉及的节点外围生成一个带手绘感的虚线“循环区域”容器，让逻辑层次一目了然。

## 【面向开发者】

### v0.4.0 连线系统架构实现与循环检测
1. **多线型 Edge 组件化**：基于 `@xyflow/react` 实现了 `SequentialEdge`, `ConditionalEdge` 和 `LoopEdge`。通过自定义 SVG Filter 实现了基于 Edge ID Seed 的动态手绘笔触（Pencil Filter）。
2. **Click-to-Connect 状态机**：在 Zustand Store 中引入了 `clickConnectState` 状态机，解决了连续点击事件中的端点捕获与高亮脉冲动画同步问题。
3. **边缘重连 (Edge Reconnection)**：支持了 `onReconnect` 钩子，具备 25px 的磁吸半径，提升了编辑效率。
4. **LoopRegionNode 自动管理**：开发了 `useLoopRegion` Hook，监听 `edges` 数组变化。通过计算 source/target 节点的 Bounding Box，动态产生/销毁 `loop_region` 类型的背景节点，并有效规避了 `setNodes` 导致的无限重新渲染循环。
5. **手绘增强过滤器**：在全局 SVG Defs 中注入了基于 `feTurbulence` 和 `feDisplacementMap` 的滤镜组，实现了全系统风格统一的抖动纹理效果。
